# 🎠 Gallery Slider

A fullscreen image slider built with **React**, **TypeScript**, **TailwindCSS** and **GSAP**, featuring smooth slide transitions and a cinematic design.

![Preview](public/preview.jpg)

---

## ✨ Features

- 🖼️ **Fullscreen image slider** — immersive visual experience
- ⏩ **Auto-play** — advances every 5 seconds
- 🎯 **Smooth slide transitions** — clean right-to-left animation
- 🖱️ **Interactive navigation** — arrow buttons and preview cards
- 📱 **Responsive design** — adapts to any screen size
- 🎨 **Dark vignette** — cinematic edge darkening
- 🌟 **Preview bar** — thumbnail cards with active state indicator

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/sebastianvasquezechavarria1234/gallery-slider.git

# Navigate to the project
cd gallery-slider

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```

## 🎯 How It Works

1. **Auto-play** — The slider automatically advances to the next slide every 5 seconds
2. **Navigation** — Use the ← → arrows or click any preview card to jump to a specific slide
3. **Transition** — Each slide change uses a smooth right-to-left animation powered by GSAP
4. **Loop** — The slider loops back to the first image after the last one
